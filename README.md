# Streamlit Portfolio — William Mohlahlane

A simple, modern **Streamlit portfolio web app** with a sidebar menu and styled “card” sections for Profile, Education, Experience, and Contact details.

## Features
- **Sidebar navigation** (Profile, Education, Experience, Contact)
- Clean **card-based UI** with custom CSS styling
- Quick-to-edit content (name, skills, education, experience)
- Clickable **email / LinkedIn / GitHub** links

## Built With
- **Python**
- **Streamlit**
- **Pandas / NumPy** (imported for data support and future extension)

## Prerequisites
- Python 3.9+ recommended
- pip (comes with Python)
